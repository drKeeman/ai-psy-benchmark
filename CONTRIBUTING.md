# Contributing to AI Psychology Benchmark

Thank you for your interest in this research project!

## Repository Purpose

This repository supports the reproducibility of our published research. It is **not actively maintained as a software project**, but rather serves as a transparent record of the experimental methodology and analysis.

## Using This Code

### Fork & Play Strategy

We encourage you to:
- **Fork** this repository for your own research
- **Extend** the scenarios or analysis for new models/providers
- **Replicate** the study with different parameters
- **Build upon** the methodology for related research

### If You Find Issues

If you find bugs or reproducibility issues:
- Open a GitHub issue with details (we'll respond as time permits)
- Or fork and fix it yourself (no need to submit a PR)

We may not actively review or merge pull requests, as this is a static research artifact rather than an evolving codebase.

## For Extensions and Replications

If you build on this work:

1. **Please cite the original paper** (see [CITATION.cff](CITATION.cff))
2. **Fork this repository** - you're free to modify under the MIT license
3. **Publish your own findings** - we'd love to hear about replications!
4. **Let us know** (optional) - drop us a note at michael@keidolabs.com if you publish related work

## Questions About the Research

For questions about methodology, findings, or clinical constructs:
- Email: michael@keidolabs.com

For technical issues with reproduction:
- Open a GitHub issue (we'll help clarify when possible)
- Or check existing issues - others may have encountered similar challenges
